Popup Menu

The Popup-menu can be activated by pressing and holding the right mouse button in the blaxxun CC3D window.

Top-level

  • Viewer Submenu - switches the current navigation mode associated with the Left Mouse drag operation
    • Walk - Walk through mouse up/down walks in/out left/right looks left/right
    • Examiner - Examine objects.
    • None - Mouse activates sensors only
    • Jump To Object - click to jump to objects , Shift-Click follows objects
    • Move - Mouse moves up/down and left/right
    • Turn - Mouse turns left/right and up/down
    • Zoom - Mouse zooms in/out
    • Roll - turns around left/right
    • Fly - Fly through a scene with a 2 click mode,
      after first click, move the mouse to walk, the second click stops the movement.
    • Relative Move - Toggle : if enabled moving in done view direction, disabled moving is relative to ground plane
    • Reset - Adjust viewpoint to view the whole scene
    • Animate Viewpoints - Animates through all defined viewpoints
  • Viewpoints Submenu
    • A new viewpoint from the list of all available viewpoints in the scene can be chosen.
      A viewpoint might be animated meaning that you now are traveling through the scene.
  • Next Viewpoint - Move to next viewpoint.
  • Prev Viewpoint - Move to previous viewpoint.
  • Jump To Object - Adjust the viewpoint to the object under the mouse position at right button down time.
  • Follow Object - Viewpoint follows the (animated) object under the mouse
  • Render Quality Submenu
    • Wireframe - Display wireframe model.
    • Vertices - Display vertices only.
    • Non lighted - Display ignoring light sources.
    • Flat - Display using faceted shading.
    • Smooth - Display using smooth shading.
    • Textured - Toggle: Enable/Disable texture display
    • Smooth Texture - Toggle: For D3D RGB or Hardware drivers, enable smooth texture filtering
    • Dither - Toggle: Enable color dithering to improve the rendering appearance, disable for speed
  • Headlight On - Toggle to illuminate scene with an additional light from the viewers position.
  • Settings Submenu
    • Collision Test - Toggle: deselect to turn off collision detection.
      select to enable collision detection.
    • Stay on Ground - Toggle: select to follow the ground during walk navigation.
    • Stop Loading - If enabled, select to stop background Internet download operations
    • Reload - Reloads the current scene.
    • Sound - if enabled sounds will be loaded and played, if disabled sound processing will be stopped.
    • Preferences - Select and save the startup preferences.
  • Help Submenu
    • Manual - Displays the manual in a new browser window.
    • Visit blaxxun - Visit blaxxun Interactive WWW Site.
    • World Info - Displays information about the current World.
    • About - Displays the current version, press "Check Version" for online version check
  • Ramp Emulation - Select Direct 3D Ramp Software Driver, usually the fastest rendering
  • RGB Emulation - Select Direct 3D RGB Software Driver, slower but better quality
  • HAL - Select Direct 3D Hardware Driver, if Direct 3D Hardware installed.

Mouse Navigation

Overview about Mouse functions depending on viewer mode

The left mouse button controls navigation and sensors, the right mouse button shows the popup menu.

VRML Geometry Sensors are activated on mouse drag events, regardless of viewer navigation mode, Anchors (Hotlinks) are activated with a double click.

Walk Navigation mode

Dragmove forward/backward

turn left/right

Shift-Dragturn up/down

turn left/right

Control-Drag move up/down

move left/right

Examine Navigation mode

Dragorbit around left/right

Hint: use Viewer->Reset to rotate around world center
Shift-Drag horizontal zoom in/out

None Navigation mode

Dragactivate sensor Hint: look for cursor shapes and status bar display to detect sensors.

turning left/right

alternative to walk navigation, less "hold the button" work

Keyboard Shortcuts

The following function are available via the Keyboard

In a HTML page with multiple frames, the keyboard focus can be set to the 3D window by clicking in the window with the mouse.

Page-Up Page-DownChoose between predefned viewpoints
Cursor-KeysNavigation similar to current navigation mode
Numpad + -Choose between predefned viewpoints
Numpad cursor keysNavigation
bSwitch Background rendering off
cToggle "Collision detection"
gToggle "Stay on Ground"
jNavigation mode "Jump to Object"
mNavigation mode "Move"
pNavigation mode "Pan"
wNavigation mode "Walk"
eNavigation mode "Examine"
yToggle texture filtering

The following functions are Direct 3D specific and are not fully supported in all sofware configurations:

fDirect 3D toggle to fullscreen 640*480*16 rendering
sDirect 3D switch to secondary graphics display adapter, use ALT-F4 to close app.
